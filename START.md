---

## Step 1: Run Locally (Test First)

Open terminal in this folder and run:

```bash
npm start
```

Then open your browser:
- **Store:** http://localhost:3000
- **Admin:** http://localhost:3000/admin (admin/admin123)

---

## Step 2: Publish Online (FREE!)

Install Vercel (one-time):
```bash
npm install -g vercel
```

Deploy your site:
```bash
vercel
```

✅ **Done!** Your website is now LIVE on the internet!

---

## 📝 Must Do Before Publishing

1. **Update Contact Info**
   - Open `views/index.html`
   - Search for: `+92 21 123456789`
   - Replace with your WhatsApp number
   - Search for: `0300-1234567`
   - Replace with your JazzCash/EasyPaisa number

2. **Change Admin Password**
   - Open `server.js`
   - Find line 45: `password === 'admin123'`
   - Change to your password

3. **Test Everything**
   - Add product to cart ✅
   - Complete checkout ✅
   - Login to admin ✅
   - View orders ✅

## 💡 Quick Tips

### Adding Products:
1. Login to admin
2. Click "Add Product"
3. Fill in details
4. Upload image or use URL
5. Save

### Managing Orders:
1. Login to admin
2. Click "Orders"
3. See all customer orders
4. Update status dropdown
5. Customer gets updated

### Changing Design:
- Colors: Edit `public/css/style.css`
- Layout: Edit `views/index.html`
- Products: Edit via admin panel
